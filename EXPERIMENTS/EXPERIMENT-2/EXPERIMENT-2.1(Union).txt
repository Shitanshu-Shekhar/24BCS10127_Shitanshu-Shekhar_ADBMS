Task
Write a query using union to stack the table 'Arts' over 'Science' and output the final table
Note:
The UNION statement removes the duplicate data in the new table formed.

SOLUTION:
SELECT * FROM ARTS 
UNION 
SELECT * FROM SCIENCE;
